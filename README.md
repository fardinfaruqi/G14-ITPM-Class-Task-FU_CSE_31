For this course assignment, students were required to:

* Form a group
* Create a GitHub repository
* Add all group members as collaborators
* Create individual branches for each member
* Push their assigned tasks to their respective branches

## First Task

The first assigned task for each member is:

* Create a **Login Form** using HTML
* Create a **Signup Form** using HTML

## Second Task

The second assigned task for each member is:

* Implement CSS in both **Login Form** & **Signup Form**.

## Third Task

The third assigned task for each member is:

* Create a `/student` route that returns student info as JSON to frontend.
* Show the SS of the output when hitting the API. 

## Fourth Task

The fourth assigned task for each member is:

* Build all API endpoints for a Todo application. The required endpoints are:
  * Create Todo
  * Update Todo
  * Delete Todo
  * Get All Todos
  * Get Selective Todo (single todo by ID or based on query)
* Requirements:
  * Implement all endpoints properly.
  * Test every API endpoint using Postman or any API testing tool.
  * Add screenshots of the testing result windows to your GitHub README file.
  * Push the complete project to GitHub branch , renamed the branch with you ID only.

Each member completes the task and pushes the work to their **own branch**.
